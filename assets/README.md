# Assets Folder

Place your assets in the following structure:

## Images
- `images/profile.jpg` - Your profile photo
- `images/placeholder.jpg` - Fallback for missing project images
- `images/blog-placeholder.jpg` - Fallback for missing blog images

## Icons
- `favicon.png` - Browser tab icon (32x32 or 64x64)

## Documents
- `documents/resume.pdf` - Your downloadable resume/CV
